-----------------------

"Aplica las siguientes animaciones CSS a las imágenes decorativas:

1. Animación de flotación vertical para las imágenes laterales

Todas las animaciones deben ser suaves y no invasivas."


PROMPT 2.3: RESPONSIVE
----------------------

"Ajusta las decoraciones para que sean responsive:

1. En tablets (≤768px): Ocultar decoraciones laterales

-------------------------------

"Optimiza los efectos visuales de las decoraciones:

5. Usa z-index apropiado para cada capa

PROMPT 2.5: INTEGRACIÓN CON DISEÑO EXISTENTE
--------------------------------------------

"Integra las decoraciones con el diseño actual:

1. Usa la misma paleta de colores para las etiquetas de texto
2. Mantén la coherencia con los bordes redondeados
